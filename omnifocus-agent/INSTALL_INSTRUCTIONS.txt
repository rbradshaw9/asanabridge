AsanaBridge Installation Instructions
=====================================

IMPORTANT: macOS Security Notice
--------------------------------
This app is not notarized by Apple, so macOS will show a security warning.
This is normal and safe - follow these steps to install:

Method 1 (Recommended):
1. Drag AsanaBridge to Applications folder
2. Open Finder → Applications
3. Right-click on AsanaBridge
4. Select "Open" from the menu
5. Click "Open" when macOS asks for confirmation

Method 2 (Alternative):
1. Try to open AsanaBridge normally (it will be blocked)
2. Go to System Preferences → Security & Privacy → General
3. Click "Open Anyway" next to the AsanaBridge message
4. Click "Open" when prompted

After First Launch:
-------------------
1. The setup wizard will guide you through configuration
2. You'll need your AsanaBridge account credentials
3. The agent will connect to your account automatically

Troubleshooting:
----------------
- If you see "damaged app" error, try downloading again
- Make sure you're running macOS 10.15 (Catalina) or later
- Contact support if you continue having issues
